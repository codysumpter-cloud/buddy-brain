# Hermes/Buddy Native Skills v3

Buddy Brain mirror of the native skill map for Buddy/Hermes/KnowledgeVault coordination.

## Purpose

Buddy Brain can reason over the available skills and help route requests, but it should not directly execute external actions. Execution belongs to Buddy/Hermes adapters and explicit approval gates.

## Included capability groups

- Social content planning and draft generation.
- Modular content factory workflows.
- Read-only public social insight analysis.
- Read-only bookmark and article digests.
- Memory and KnowledgeVault synthesis.
- Pre-call context preparation.
- Responsible risk education and sports-model review workflows.

- `sports-model-risk-evaluator`

## Runtime policy

Buddy Brain may summarize, compare, and recommend skill selection. It must not publish content, contact people, write to external accounts, mutate repositories, change memory, or operate regulated-domain flows directly.
